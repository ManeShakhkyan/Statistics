# 📈 Health Insurance Forecasting & Statistics Engine

Այս նախագիծը նախատեսված է առողջության ու կյանքի ապահովագրության վճարների վերլուծության և կանխատեսման համար։ Այն միավորում է ժամանակային շարքերի (Time Series) առաջադեմ մոդելները և ռեգրեսիոն վերլուծությունը մեկ միասնական համակարգում։

## 🚀 Նախագծի Կառուցվածքը

Նախագիծը բաժանված է երկու հիմնական մոդուլի.

* **Time Series Module (`/time_series`):** Ժամանակային շարքերի կանխատեսում օգտագործելով Seasonal Random Walk, Holt-Winters և Theta մոդելները առողջության ապահովագրության եռամսյակային տվյալների հիման վրա:
* **Regression Module (`/regression`):** Գործոնային վերլուծություն և կախվածությունների ուսումնասիրություն կյանքի ապահովագրության մասին:

Theta Model: Ժամանակային շարքի քայքայում (decomposition) և SES հարթեցում:

Seasonal Random Walk: Բազային մոդել սեզոնային աճի տեմպերի հիման վրա:
